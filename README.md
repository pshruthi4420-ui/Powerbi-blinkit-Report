**Project Title**

**Blinkit Sales Data Analysis Dashboard**

**Problem Statement**

This project analyzes Blinkit grocery sales data to identify high-performing outlets, top product categories, and sales distribution across different outlet types.

**Tools Used**

Power BI

SQL

Excel (for data preprocessing)

**Key Metrics**

Total Sales: $1.20M

Average Sales: $141

Number of Items: 8523

Average Rating: 3.9

**Key Insights**

Tier 3 outlets generate the highest sales (~$472K).

Medium-sized outlets contribute the most revenue (~$507K).

Fruits and snacks are the top-performing product categories.

Supermarket Type 1 contributes the majority of total sales.
